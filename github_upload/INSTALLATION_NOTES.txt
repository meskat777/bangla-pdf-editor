BEFORE FIRST RUN
========================================

1. Install Python Dependencies:
   pip install -r requirements.txt

   Note: First run will download EasyOCR models
   (~200MB total). This happens automatically.

2. Verify Installation:
   python3 test_installation.py

========================================
TO START THE APPLICATION
========================================

Option 1 (Recommended):
   python run.py

Option 2 (Direct):
   python backend/app.py

Then open browser:
   http://localhost:5000

========================================
TROUBLESHOOTING
========================================

Server won't start:
‚Üí Run: pip install -r requirements.txt
‚Üí Check Python version (need 3.8+)

Upload fails:
‚Üí File must be PDF
‚Üí Size must be < 50MB

OCR not working:
‚Üí Wait for model download (first run only)
‚Üí Check internet connection

Fonts not showing:
‚Üí Verify fonts/ directory has .ttf/.otf files
‚Üí Restart server
